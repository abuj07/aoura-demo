# Aoura — Interactive Demo

**Live demo:** `https://[your-username].github.io/aoura-demo`

This is the interactive HTML prototype for Aoura — *Your private space to talk, reflect, and be heard.*

## What this demo shows

- 13 AI companions with distinct personalities, voices, and backgrounds
- The Family group chat — Dad, Mum, Temi and Tobi responding in their own voices
- Real friends section (Jamie, Aisha)
- Voice input (Web Speech API)
- Disappearing messages with timer selection
- Mood selector
- Contextual notifications (appear when tab is in background)
- Crisis detection with UK and US support resources
- Companion memory and insight bubbles
- Right panel — memories, traits, mood read, growth moments
- Fully mobile responsive
